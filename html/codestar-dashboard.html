CodeStar: Dashboard Tiles

Introduction

Once you've sucessfully launched a solution based on one of the CodeStar project templates, you can view the Dashboard for the project. From this dashboard, you can arrange a collection of tiles. These tiles provide at-a-glance project information and allow you to go directly to the highlighted service - such as CodeCommit, CodePipeline, and CloudWatch. Moreover, you can choose to remove any of the dashboard tiles you'd rather not see. In this sublesson, we'll make some changes and access some of the services exposed through the dashboard.

Edit the wiki tile

  • Choose the ellipses at the top right corner of the wiki tile and choose Edit.
  • After selecting Edit, you'll be presented with the Markdown editor. As is implied, the wiki tile supports Markdown - a cheatsheet for Markdown is provided here.

Commit History
