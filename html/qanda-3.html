How can you define a CodeBuild project without using a buildpsec.yml file?

  • You can't; CodeBuild must find a buildspec.yml file to properly run
  • Provide the build commands in the BuildSpec section of a CloudFormation template
  • Provide the build commands in the BuildSpec section of a CodePipeline
  • Include a Dockerfile and CodeBuild will automatically build a docker image

How does the CodeDeploy deployment group know which EC2 instances to deploy to?

  • The instance id is provided as an input to the CodeDeploy deployment group resource
  • The instance private IP is provided as an input to the CodeDeploy deployment group resource
  • The logical name of the resource in the CloudFormation template is provided as an input to the CodeDeploy deployment group resource
  • The instance tag name and value are provided as an input to the CodeDeploy deployment group resource
