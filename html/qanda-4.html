How can you use Lambda for testing your serverless application?

  • Create unit tests in the "tests" subdirectory and CodePipeline will automatically execute those tests
  • Create unit tests in the "tests" subdirectory and the Serverless Application Model will automatically execute those tests
  • Create a Lambda function that is invoked via a Lambda action in CodePipeline that performs the testing
  • Create a Lambda function that notifies QA team members via SNS to begin manual testing of the application
