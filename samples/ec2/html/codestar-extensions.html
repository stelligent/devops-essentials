CodeStar: Extensions

Introduction

You can integrate third-party tools with CodeStar using the Extensions feature. At the time of this recording, CodeStar provides support for JIRA for issue-tracking. There is every expecation that there will be additional extensions for CodeStar.

In this sublesson, you will configure access to an existing JIRA account. If you do not have JIRA, you'll at least see the process for interfacing with third-party tools in CodeStar
